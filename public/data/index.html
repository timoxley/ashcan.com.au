Ashcan Site Instructions

How to add a Comic

  • Add comics to `/comics/` as `/comics/1/`, `/comics/2/`, `/comics/n/`, etc
  • Add pages to `/comics/n/pages/1.jpg`, `/comics/n/pages/2.jpg`, `/comics/n/pages/3.jpg`
  • Add comic number in `/index.json` to the 'comics' array.

Example comic JSON file

  date: "30/2/2012",
  description: "Markdown",
  price: "5",
  stories: [

How to add News

  • Add markdown file to /news/ as `/news/1.md`, `/news/2.md`, `/news/n`, etc
  • Add news number in `/index.json` to the 'news' array.

How to upload

  • Zip site up.
  • Drop on staticloud.com
